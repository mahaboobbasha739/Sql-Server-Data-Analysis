					SYNONYMS
					--------
	SYNONYMS ARE USED TO HIDE THE ACTUAL TABLE NAMES
	SYNONYMS IS ALSO KNOWN AS ALIAS_NAMES
	SYNONYMS ALLOWS DML OPERATIONS
	SYNONYMS EXISTS PHYSICALLY IN THE DATABASES
	SYNONYMS ARE USED FOR SECURITY
	

	SELECT * FROM EMP		-- ONE PART NAME
	SELECT * FROM DBO.EMP		-- TWO PART NAME
	SELECT * FROM SYNONYMS.DBO.EMP	-- THREE PART NAME

	------------------------------------------------------------------

	DIFFERENCE B/W VIEW AND SYNONYM
	--------------------------------

	VIEW
	-----
	VIEW IS VIRTUAL TABLE IT DOESN'T OCCUPY ANY SPACE IN DATABASE
	COLUMN & ROW LEVEL SECURITY
	VIEWS CAN BE CREATED ON MULTIPLE TABLES


	SYNONYMS
	---------
	SYNONYMS EXISTS PHYSICALLY IN THE DATABASE AND IT OCCUPY SOME SPACE IN DATABASE
	NOT POSSIBLE TO CREATE THE COLUMN & ROW LEVEL SECURITY
	SYNONYMS ARE CREATED ON ONLY SINGLR TABLE
